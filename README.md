# 1 Крестики-нолики (Tic‑Tac‑Toe)

Игра с графическим интерфейсом на Pygame.  
Вы – крестики, компьютер – нолики.

## Особенности
- Два режима сложности: **Лёгкий** (случайные ходы) и **Сложный** (минимакс).
- Удобное управление: клик по клетке, кнопка «Новая игра» или клавиша `R`.
- Аккуратный дизайн, эффект наведения на кнопки.

  # 2 Тест скорости печати на английском(Typing Speed Test) 

Игра для тренировки навыков быстрого и точного набора текста на клавиатуре.  
Вам показывается случайное предложение, которое нужно напечатать как можно быстрее и с меньшим количеством ошибок.

## Особенности

- Случайные предложения из встроенного списка (можно легко расширить).
- Расчёт **времени**, **точности** (%), **количества введённых символов** и **слов в минуту (WPM)**.
- Простой и понятный интерфейс.
- Кнопка **Reset** для повторного прохождения теста.
- Приятный сюрприз для друга)

##  Управление

1. **Кликните** по полю ввода – начнётся тест.
2. **Печатайте** текст – введённые символы отображаются в поле.
3. Нажмите **Enter** – тест завершится, и вы увидите результаты.
4. После завершения кликните по надписи **Reset** (внизу экрана) или перезапустите игру, чтобы начать новый тест.

## Запуск 1
```bash
pip install pygame
python main.py
Управление
Клик – ход.
```

# 3. Todo CLI — консольный менеджер задач на Python

Простое и удобное CLI-приложение для ведения списка дел. Данные хранятся в локальном файле `todo.txt`, а управление происходит через интуитивные команды.

##  Возможности

-  Добавление задач с автоматической генерацией ID
-  Просмотр всех текущих задач
-  Завершение (удаление) задачи по ID
-  Автоматическое сохранение в файл `todo.txt`
-  Простой и понятный интерфейс командной строки

## Установка

1. **Клонируйте репозиторий** (или просто скачайте файл `todo.py`)
   ```bash
   git clone https://github.com/ваш-логин/todo-cli.git
   cd todo-cli
Установите зависимость — библиотеку click:

pip install click
(Если у вас несколько версий Python, используйте pip3 или python -m pip install click)

  Использование
Все команды выполняются через основной скрипт. Например:

Команда	Описание
tasks	Показать все текущие задачи
add	Добавить новую задачу
done	Завершить (удалить) задачу по ID
+ Добавление задачи
bash
python todo.py add
После запуска программа попросит ввести текст задачи.
Можно также передать текст сразу через опцию -add:

bash
python todo.py add -add "Купить продукты"
Пример вывода:

text
 Добавлена задача "Купить продукты" с ID 0
Просмотр всех задач
bash
python todo.py tasks
Вы увидите список всех незавершённых задач с их ID.

Пример вывода:

text
ВАШИ ЗАДАЧИ
**********
• Купить продукты (ID: 0)
• Позвонить маме (ID: 1)
• Написать отчёт (ID: 2)
 Завершение задачи
bash
python todo.py done
Введите ID задачи, которую хотите удалить.
Или укажите ID сразу через опцию -fin:

bash
python todo.py done -fin 1
Пример вывода:

text
✅ Завершена и удалена задача "Позвонить маме" с ID 1
