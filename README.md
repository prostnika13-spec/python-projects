# python-projects
1 # Крестики-нолики (Tic‑Tac‑Toe)

Игра с графическим интерфейсом на Pygame.  
Вы – крестики, компьютер – нолики.

## Особенности
- Два режима сложности: **Лёгкий** (случайные ходы) и **Сложный** (минимакс).
- Удобное управление: клик по клетке, кнопка «Новая игра» или клавиша `R`.
- Аккуратный дизайн, эффект наведения на кнопки.

## Запуск
```bash
pip install pygame
python main.py
Управление
Клик – ход.

R или кнопка «Новая игра» – перезапуск.

Кнопки «Лёгкий»/«Сложный» – смена режима (активный подсвечен).

Технологии
Python 3.7+, Pygame
