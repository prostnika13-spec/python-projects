# Тест скорости печати на английском(Typing Speed Test) 

Игра для тренировки навыков быстрого и точного набора текста на клавиатуре.  
Вам показывается случайное предложение, которое нужно напечатать как можно быстрее и с меньшим количеством ошибок.

## Особенности

- Случайные предложения из встроенного списка (можно легко расширить).
- Расчёт **времени**, **точности** (%), **количества введённых символов** и **слов в минуту (WPM)**.
- Простой и понятный интерфейс.
- Кнопка **Reset** для повторного прохождения теста.
- Приятный сюрприз для друга)

##  Управление

1. **Кликните** по полю ввода – начнётся тест.
2. **Печатайте** текст – введённые символы отображаются в поле.
3. Нажмите **Enter** – тест завершится, и вы увидите результаты.
4. После завершения кликните по надписи **Reset** (внизу экрана) или перезапустите игру, чтобы начать новый тест.
